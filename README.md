# Telegram-бот для оценки эффективности рекламных каналов 📊

Этот проект представляет собой аналитического Telegram-бота, который автоматизирует расчет маркетинговых метрик, строит графики и делает прогнозы с помощью машинного обучения.

## Основной функционал
* Автоматический расчет метрик: CTR, CPC, CPL, CPA, ROI, ROAS.
* База данных SQLite: cохранение истории аналитики по каждому пользователю.
* Визуализация (Matplotlib & Seaborn): uенерация столбчатых диаграмм окупаемости и матриц корреляций.
* Машинное обучение: gрогнозирование количества лидов на основе заданного бюджета с использованием модели `LinearRegression`.

## Стек технологий
* Python 3.x
* aiogram 3.x (Асинхронный Telegram API)
* Pandas, NumPy (Обработка данных)
* Scikit-learn (Машинное обучение)
* Matplotlib, Seaborn (Визуализация)
* SQLite (База данных)
